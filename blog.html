These instructions are for Photoshop CS5 on Mac and Windows. Setting up CS4 is very similar. Step 1: Go to Edit → Color Settings and set the working space for RGB to Monitor RGB. Step 2: Open a document and go to Edit → Assign Profile, then set it to Working RGB. This must be done for every single document you work on. Step 3: Ensure View → Proof Colors is turned off. Step 4: When saving files with Save for Web & Devices, ensure that Convert to sRGB is turned off. If youre saving a JPEG file, then also turn off Embed
